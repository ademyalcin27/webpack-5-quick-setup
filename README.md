Webpack Quick Setup

## This project for quick setup for your starter project

[You can watch video](https://studio.youtube.com/video/tD33Ve6rxAU/edit)

## Project setup
```
npm install
```

### Compiles and hot-reloads for development
```
npm run start
```

### Compiles and minifies for production
```
npm run build
```
